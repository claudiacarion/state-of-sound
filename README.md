# 🎵 State of Sound - React State Project

State of Sound is a personal music web app built with React that showcases the artists I keep on repeat and the new discoveries I’ve recently added to my rotation. The site is a Single Page Application (SPA) with dynamic content and background images, demonstrating React concepts like state, props, and component reuse.

## ✨ Features

- Single Page Application (no routing) with "pages" implemented with React state:
- Home, On Repeat, Discovery, and About pages
- Dynamic backgrounds depending on the page
- Reusable components
- Responsive design for mobile and desktop
- Images for each page and artist
- Clean, modular CSS with CSS variables

## 👩‍💻 Technologies

- React (functional components, props, state)
- CSS / CSS Variables / CSS Modules
- Responsive design
- Deployment with Vercel
